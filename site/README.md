# Nucers前端工程

> 本项目依赖`yarn`进行包管理，可以使用`yrm`切换软件源，`yrm use taobao`可以使用淘宝源

## Getting Started

开发环境启动

```bash
npm run dev
# or
yarn dev
```

浏览器打开[http://localhost:3000](http://localhost:3000)查看结果
